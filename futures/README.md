# 期货综合分析系统

## 黄金综合评分公式

### 公式结构
```
S = w₁·T + w₂·O + w₃·R

其中:
T = 技术面得分 (权重 25%)
O = 原油/通胀得分 (权重 20%)
R = 避险情绪得分 (权重 55%)
```

### 避险情绪 R 包含
| 子因素 | 权重 | 利好黄金条件 |
|-------|------|-------------|
| 地缘政治 G | 25% | 战争/冲突/制裁 |
| VIX恐慌指数 | 20% | VIX > 20 |
| 美元指数 | 15% | DXY < 95 (弱美元) |
| 加密货币 | 15% | BTC 下跌 |
| 美债收益率 | 15% | 收益率下跌 |
| 信用利差 | 10% | 利差扩大 (>400bp) |
